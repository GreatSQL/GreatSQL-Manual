# 认证资质与荣誉

## 认证资质

GreatSQL 数据库是开放原子开源基金会旗下捐赠项目，拥有 中国信通院 **可信开源社区** 及 **可信开源项目** 双认证。

GreatSQL 数据库同时也是gitee（码云）平台上的 GVP 项目，详见：[https://gitee.com/gvp/database-related](https://gitee.com/gvp/database-related) **数据库相关**类目。
