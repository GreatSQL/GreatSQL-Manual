# EXPLAIN PLAN FOR
---

```
EXPLAIN PLAN FOR
  [STATMENT]
```

该语法仅为了Oracle语法能够兼容，实际功能与原生的EXPLAIN一样。

与Oracle区别：会直接显示EXPLAIN结果。
