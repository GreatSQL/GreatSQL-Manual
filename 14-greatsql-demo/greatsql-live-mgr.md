# 关于 GreatSQL Live MGR

GreatSQL Live MGR 是包含 3 个节点的 MGR 在线测试体验环境，并且还可以提供通过 GreatSQL Shell 来管理 MGR 集群。

::: tip 提示
最多支持 20 人同时在线体验测试。

完整阅读使用说明，即可开始在线体验。
:::
